 # 反函数的求导法则
- 如果函数$x=f(y)$在区间$I_{y}$内单调、可导且$f'(y)\neq 0$，那么它的反函数$y=f^{-1}(x)$在区间$I_{x} = {x|x=f(y),y\in I_{y} }$内也可导，且$$[f^{-1}(x)]'= \frac{1}{f'(y)}\text{或}\frac{ dx }{ dy } =\frac{1}{\frac{ dx }{ dy } } $$
