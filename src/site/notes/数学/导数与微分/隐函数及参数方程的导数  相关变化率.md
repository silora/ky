# 参数方程确定导数
- 参数方程$\begin{cases} x=\varphi(t)\\y=\psi(t)\end{cases}$的一阶导数为$\frac{ dy }{ dx }= \frac{\psi'(t)}{\varphi'(t)}$，二阶导数为$\frac{ d^{2}y }{ dx^{2} }=\frac{ {\psi''(t)\varphi'(t)-\psi'(t)\varphi''(t)} }{\varphi'^3(t)}$
