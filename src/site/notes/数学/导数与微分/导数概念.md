# 定义
- $\lim{x\to x_{0} } \frac{ {f(x)-f(x_{0})} }{x-x_{0} } = \lim_{\Delta x\to0} \frac{\Delta y}{\Delta x}$
- 单侧导数
	- 如果$f(x)$在开区间$(a,b)$可导，且$f'_{+}(a)$，$f'_{-}(b)$都存在，那么就说$f(x)$在$[a,b]$可导
